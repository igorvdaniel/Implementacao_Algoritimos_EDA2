# Implementação de Algorítmos de Ordenação

- Este repositório tem como objetivo Implementar algoritimos de ordenação diversos, com suas variações e tentando sempre aplica-los à contextos orientados a objetos.

## Busca Sequencial
- Foi aplicado a Busca Sequencial Normal e a com Sentinela, onde no arquivo em C podemos ver a diferença, mesmo que mínima, de ganho de desempenho entre as duas implementações. Sendo mais vísiveis com entradas na casa dos milhões.
- Além disso foi possível aprender algo muito significativo em relação ao C#, por mais que um código possa parecer mais simples, ele pode não ser o mais otimizado, neste caso a lição específica é que retornar NULL é mais rápido que retornar um Objeto.
